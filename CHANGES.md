# Опис змін у файлі LSTM_UKF_St_IMM.py

## Проблема, що вирішується
Виявлено проблему з довірчими інтервалами в моделі LSTM_UKF_St_IMM: 
- Фактичне покриття ДІ: 56.79% (замість цільового 85.24%)
- Середня ширина ДІ / stddev: 8.00x
- Потрібно було розширити довірчі інтервали для кращого покриття

## Внесені зміни

### 1. Додано новий метод `_calibrate_confidence_interval_fixed`

Цей метод реалізує покращену логіку для калібрування довірчих інтервалів:

#### Зміни параметрів:
- Розширені межі для z-квантилів: `z_lower = tf.clip_by_value(z_lower_raw, -7.0, -0.1)` та `z_upper = tf.clip_by_value(z_upper_raw, 0.1, 7.0)`
- Збільшено максимальну ширину довірчих інтервалів: `max_width_factor = 10.0 + 5.0 * regime_scale_factor`
- Змінено параметри Student-t розподілу для товстих хвостів: `dof_base = 2.5`, `tail_weight_pos = 0.4`, `tail_weight_neg = 2.0`

#### Покращення асиметрії:
- Знижено поріг екстремальної волатильності до 0.6
- Посилено асиметрію для широкого діапазону даних: `lower_expansion_factor = 1.0 + 5.0 * extreme_vol_mask * volatility_level`
- Додано аналіз асиметрії інновацій з відповідною корекцією

#### Покращення прямої корекції за покриттям:
- Збільшено агресивність корекції для сигналів з високою волатильністю
- Додано додаткову асиметричну корекцію при великій похибці покриття

### 2. Змінено виклики методів

Замінено виклики `_calibrate_confidence_interval` на `_calibrate_confidence_interval_fixed` у двох місцях:
- Рядок 2376 у функції `train_step`
- Рядок 3207 у функції оцінки

### 3. Особливості реалізації

- Усі зміни сумісні з графовим режимом TensorFlow
- Використовуються лише функції, сумісні з `@tf.function`
- Забезпечено правильне керування розмірностями тензорів
- Додано перевірки на валідність параметрів

## Очікуваний ефект

- Покращення фактичного покриття довірчих інтервалів до близьких до цільових 85%
- Збільшення ширини інтервалів для кращого охоплення екстремальних значень
- Краща адаптація до високоволатильних ринків з широким діапазоном значень
- Збереження обчислювальної ефективності моделі
